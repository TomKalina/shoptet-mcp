# Create invoice from order

> "Create an invoice for order #2024000456"

Uses `create_invoice_from_order` to generate an invoice document from an existing order.

## What you can ask

- "Generate an invoice for order #456"
- "Create invoices for all completed orders from last week"
- "Show me the invoice PDF for order #789"

## Tools used

- `create_invoice_from_order` - generate invoice from order code
- `get_invoice_pdf` - download invoice as PDF
- `list_invoices` - list all invoices

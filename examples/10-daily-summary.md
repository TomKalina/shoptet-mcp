# Daily eshop summary

> "Give me a summary of today's activity"

Combines multiple tools to create a comprehensive daily report.

## What you can ask

- "How did my shop do today?"
- "Daily summary report"
- "How many orders and revenue today?"
- "Weekly performance overview"

## Tools used

- `list_orders` - today's orders with totals
- `get_order_statuses` - status breakdown
- `list_products` - product count
- `get_eshop_info` - shop metadata
- `get_order_changes` - recent activity

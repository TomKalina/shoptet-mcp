# Track product changes for catalog sync

> "Show me product changes from last week"

Uses `get_product_changes` to list recently modified products — useful for syncing with external systems.

## What you can ask

- "What products changed since Monday?"
- "Show me new products added this month"
- "Which products were updated today?"

## Tools used

- `get_product_changes` - list changed products since a date
- `get_order_changes` - list changed orders since a date

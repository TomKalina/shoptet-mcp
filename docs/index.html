Upozornění — použití na vlastní nebezpečí

Tento nástroj dává AI asistentovi plnou kontrolu nad vaším eshopem prostřednictvím Shoptet API. AI může vytvářet, upravovat a mazat produkty, objednávky, zákazníky a další data. Před použitím si důkladně zvažte, jaká oprávnění vašemu API tokenu přidělíte. Doporučujeme nejprve vyzkoušet na testovacím eshopu. Autoři nenesou žádnou odpovědnost za případné škody způsobené použitím tohoto nástroje.

Co všechno umí?

48 nástrojů pro kompletní správu vašeho Shoptet eshopu přímo z AI asistenta.

📦

Objednávky

Vytvářejte, prohlížejte a aktualizujte objednávky. Měňte stavy, přidávejte poznámky, generujte PDF.

list_orders get_order create_order update_order update_order_status
🎁

Produkty

Plná CRUD správa produktového katalogu. Vytvářejte, upravujte, mažte produkty a spravujte kategorie a značky.

create_product update_product delete_product get_stock list_categories
👥

Zákazníci

Vyhledávejte, vytvářejte a aktualizujte zákaznické profily s kontaktními údaji a adresami.

list_customers get_customer create_customer update_customer
📄

Doklady

Faktury, proforma faktury, dobropisy, dodací listy a doklady o zaplacení — vše na jednom místě.

create_invoice_from_order get_invoice_pdf create_credit_note create_delivery_note
🔗

Webhooky

Nastavte si real-time notifikace o událostech na eshopu — nové objednávky, změny produktů a další.

create_webhook list_webhooks delete_webhook
📊

Eshop info

Základní informace o eshopu, stavy objednávek, AI prompty pro analýzu dat a denní reporty.

get_eshop_info get_order_statuses

Ukázky použití

Prostě se zeptejte přirozeným jazykem — česky nebo anglicky.

"Ukaž mi poslední objednávky"
Zobrazí seznam posledních objednávek se stavem, zákazníkem a celkovou částkou.
"Změň stav objednávky #123 na odesláno"
Aktualizuje stav objednávky. AI si samo najde správné ID stavu.
"Vystav fakturu k objednávce #456"
Vygeneruje fakturu z existující objednávky a vrátí odkaz na PDF.
"Jaké mám produkty a co je skladem?"
Projde produktový katalog a zkontroluje skladové zásoby.
"Doplň informace o whisky z Whiskybase"
AI vyhledá detaily na webu a obohatí popisy produktů o hodnocení a degustační poznámky.
"Přidej nový produkt: Lagavulin 16, cena 3500 Kč"
Vytvoří nový produkt s popisem, kategorií a cenou.
"Vytvoř webhook na nové objednávky"
Zaregistruje callback URL pro real-time notifikace o nových objednávkách.
"Dej mi denní přehled eshopu"
Kombinuje více nástrojů pro kompletní denní report — objednávky, tržby, stav skladu.
"Najdi zákazníka jan@example.com"
Vyhledá zákazníka podle emailu a zobrazí kompletní profil s historií.
"Které produkty se změnily za poslední týden?"
Sleduje změny v katalogu — užitečné pro synchronizaci s externími systémy.

Instalace

Žádná instalace — stačí nakonfigurovat váš AI klient s npx shoptet-mcp.

claude mcp add shoptet \
  -e SHOPTET_API_TOKEN=your-token \
  -- npx shoptet-mcp
{
  "mcpServers": {
    "shoptet": {
      "command": "npx",
      "args": ["shoptet-mcp"],
      "env": {
        "SHOPTET_API_TOKEN": "your-token"
      }
    }
  }
}

Kde získat API token?

1. Přihlaste se do administrace vašeho Shoptetu
2. Přejděte do Nastavení → API přístup
3. Vytvořte nový API token s požadovanými oprávněními
4. Zkopírujte token a vložte místo your-token v konfiguraci výše
